Built entirely with **HTML5, CSS3, and JavaScript**, PARYATAK requires no backend server, making it lightweight and easy to deploy. The website features a modern, responsive design with smooth animations and interactive components that enhance user engagement.

## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git (optional, for cloning)

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/manii3m/PARYATAK.git
   cd PARYATAK
   ```

2. **Open in Browser**
   
   **Option A: Direct File Access**
   ```bash
   # On macOS/Linux
   open index.html
   
   # On Windows
   start index.html
   ```
   
   **Option B: Using a Local Server** (Recommended)
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Python 2
   python -m SimpleHTTPServer 8000
   
   # Using Node.js (with http-server)
   npx http-server
   ```
   Then open `http://localhost:8000` in your browser.

   **Option C: Visual Studio Code Live Server**
   - Install the "Live Server" extension
   - Right-click on `index.html` and select "Open with Live Server"
